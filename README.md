Проект демонстрирует работу с библиотеками Pandas и SciPy: загрузку данных из CSV-файла и статистический анализ пользовательского поведения на сайте за декабрь 2024 года. 
В рамках задания рассчитаны основные описательные статистики, коэффициент вариации времени нахождения на сайте, а также выполнена проверка статистической значимости различий в количестве просмотренных страниц между пользователями мобильных и десктопных устройств.
